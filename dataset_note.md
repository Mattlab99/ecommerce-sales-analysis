## Dataset Information

This project uses the Brazilian E-commerce Public Dataset by Olist.

The dataset contains real-world e-commerce transactional data including:

- Orders
- Order Items
- Payments
- Products
- Customers
- Sellers
- Delivery timestamps

Due to GitHub file size limitations, the full dataset is not included in this repository.

You can access and download the dataset from Kaggle:
https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

This project demonstrates an end-to-end data analytics workflow, including:
- SQL-based data extraction and aggregation
- Data cleaning and transformation using Python (Pandas)
- Business KPI analysis
- Dashboard development in Power BI

The focus of this repository is on analytical thinking, data processing, and business insight generation rather than raw data storage.
